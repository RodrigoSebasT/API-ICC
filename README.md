## Especificación del API

- ### Volumen de un cubo:
    #### URL
    figurageometrica/cube/volumen/s

    #### Parametros
    - s: Lado del cubo

    #### Salida
    Volumen del cubo

    #### Ejemplo

    - URL: localhost:9000/cube/volumen/3
  
    ![https://raw.githubusercontent.com/RodrigoSebasT/API-ICC/main/static/cuboReadmeVolumen.png]()
